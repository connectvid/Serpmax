### Code Example

```javascript
const response = await fetch('https://api.serpmax.com/search', {
  method: 'POST',
  headers: {
    'X-API-Key': 'your_api_key',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    q: 'best restaurants in NYC',
    num: 10
  })

const data = await response.json();
```
